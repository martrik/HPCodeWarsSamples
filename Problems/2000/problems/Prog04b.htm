Histogram
Problem 4 Division II 4 Points
Problem Statement
A histogram is a bar graph in which the area of each bar is proportional to an item's frequency of occurrence.  Write a program to generate a histogram of the letters (A-Z) in a file.
Program Input
A collection of strings. The input file PROG04.IN will not contain any blank lines.
THE QUICK BROWN FOX JUMPS OVER THE LAZY DOG
THIS IS AN EXAMPLE OF HOW TO TEST YOUR
HISTOGRAM PROGRAM.  YOU
CAN USE THIS EXAMPLE.
Program Output
The program must display on screen a vertical histogram representing the occurrences of each letter of the alphabet. All letters will be uppercase (no need to check), and your program may ignore numbers, spaces, punctuation, etc. Lines will be no longer than 80 characters, and if a letter occurs more than 32 times the histogram should stop at 32.
